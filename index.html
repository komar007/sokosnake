sokosnake by komar007

A logic game - sokoban and snake in one

Sokosnake is a simple logic game. The player controls a snake, which pushes diamonds to so-called rooms. The goal of the game is to put all diamonds inside rooms. Though simple, it is not easy - except for diamonds and rooms there are rocks, gates, holes, teleports and other elements the player has to use carefully to pass the level. Currently the game runs in ncurses mode, but future versions will have a cross-platform Qt interface.

Dependencies

python

Install

Clone the repository or unpack the tarball, and run: 

python curses_interface.py level
where level is one of the .ss files in the levels subdirectory.
